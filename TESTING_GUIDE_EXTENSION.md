## 🐛 常见问题排查

### 问题1：数据库脚本执行失败

**错误信息**：
```
Table 'sys_announcement' already exists
```

**解决方案**：
```sql
-- 先删除所有新表
DROP TABLE IF EXISTS sys_announcement;
DROP TABLE IF EXISTS student_goal;
DROP TABLE IF EXISTS achievement_badge;
DROP TABLE IF EXISTS student_badge;
DROP TABLE IF EXISTS sys_message;
DROP TABLE IF EXISTS sys_config;
DROP TABLE IF EXISTS sys_backup;
DROP VIEW IF EXISTS v_class_sport_stats;

-- 重新执行脚本
```

---

### 问题2：操作日志页面空白

**可能原因**：
1. `operation_log` 表不存在
2. Thymeleaf模板解析错误

**解决方案**：
```bash
# 检查表是否存在
mysql -u root -p -e "USE sport_evaluation; SHOW TABLES LIKE 'operation_log';"

# 查看应用日志
tail -f logs/application.log
```

---

### 问题3：富文本编辑器不显示

**可能原因**：
CDN资源加载失败

**解决方案**：
1. 检查浏览器控制台是否有404错误
2. 替换为国内CDN：
```html
<!-- 替换前 -->
<link href="https://cdn.bootcdn.net/ajax/libs/quill/1.3.7/quill.snow.min.css" rel="stylesheet">
<script src="https://cdn.bootcdn.net/ajax/libs/quill/1.3.7/quill.min.js"></script>

<!-- 替换后（使用unpkg） -->
<link href="https://unpkg.com/quill@1.3.7/dist/quill.snow.css" rel="stylesheet">
<script src="https://unpkg.com/quill@1.3.7/dist/quill.min.js"></script>
```

---

### 问题4：公告发布后前端看不到

**可能原因**：
1. 公告状态不是"已发布"
2. 有效期设置错误
3. 缓存问题

**解决方案**：
```sql
-- 检查公告状态
SELECT id, title, status, start_time, end_time FROM sys_announcement ORDER BY create_time DESC LIMIT 5;

-- 手动修正状态
UPDATE sys_announcement SET status = 1 WHERE id = ?;
```
